# Note App

A lightweight note-taking application to create, edit, and manage personal notes directly in the browser.

## Features
- Add, edit, and delete notes
- Notes saved using browser local storage
- Simple and clean interface

## What I Learned
- Using localStorage to persist data
- CRUD operations in JavaScript
- DOM manipulation for dynamic content
- Structuring small apps logically

## Technologies Used
- HTML
- CSS
- JavaScript
